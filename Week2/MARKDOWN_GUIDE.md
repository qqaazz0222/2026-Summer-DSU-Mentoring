# 마크다운(Markdown) 문법 가이드

> `DESIGN_TEMPLATE.md` 작성 전에 참고. 마크다운은 `.md` 확장자 텍스트 파일에 기호로 서식을 표현하는 문법.
> VSCode에서 `.md` 파일 열고 우측 상단 미리보기 버튼(또는 `Ctrl+Shift+V` / Mac `Cmd+Shift+V`)으로 결과 확인 가능.

---

## 1. 제목 (Heading)

`#` 개수로 제목 크기 결정. `#` 1개가 가장 큼, 최대 6개.

```markdown
# 제목 1
## 제목 2
### 제목 3
```

결과:

# 제목 1
## 제목 2
### 제목 3

---

## 2. 강조 (굵게/기울임)

```markdown
**굵게**
*기울임*
~~취소선~~
```

결과: **굵게** / *기울임* / ~~취소선~~

---

## 3. 목록

**순서 없는 목록**: `-` 또는 `*`

```markdown
- 항목 1
- 항목 2
  - 하위 항목 (앞에 스페이스 2칸)
```

- 항목 1
- 항목 2
  - 하위 항목

**순서 있는 목록**: 숫자 + `.`

```markdown
1. 첫번째
2. 두번째
```

**체크박스** (할 일 목록, `DESIGN_TEMPLATE.md`의 과제 항목에서 사용):

```markdown
- [ ] 아직 안 한 것
- [x] 완료한 것
```

- [ ] 아직 안 한 것
- [x] 완료한 것

---

## 4. 링크 & 이미지

```markdown
[표시될 글자](https://example.com)
![대체 텍스트](이미지경로.png)
```

- 링크: `[ ]`에 보여줄 글자, `( )`에 실제 주소
- 이미지: 링크 앞에 `!` 붙임, `( )`에 이미지 파일 경로 또는 URL

## 5. 코드

**한 줄 코드**: 백틱(`` ` ``)으로 감싸기 → `` `이렇게` ``

**코드 블록**: 백틱 3개(` ``` `)로 위아래 감싸기, 언어 이름 표시 가능

````markdown
```html
<h1>안녕</h1>
```
````

결과:

```html
<h1>안녕</h1>
```

---

## 6. 표 (Table)

`|`로 칸 구분, 2번째 줄은 구분선(`---`).

```markdown
| 항목 | 설명 |
|---|---|
| id | 고유 번호 |
| title | 제목 |
```

결과:

| 항목 | 설명 |
|---|---|
| id | 고유 번호 |
| title | 제목 |

> 표 안 `|` 개수와 줄 맞출 필요 없음. 각 줄에 `|`만 잘 넣으면 자동 정렬됨.

---

## 7. 인용구 (Blockquote)

`>` 로 시작.

```markdown
> 참고: 여기는 강조하고 싶은 문장
```

> 참고: 여기는 강조하고 싶은 문장

---

## 8. 구분선

`---` 한 줄만 단독으로 (앞뒤로 빈 줄 필요).

```markdown
내용1

---

내용2
```

---

## 9. 자주 하는 실수

- 코드블록 여닫는 백틱 개수 안 맞음 (3개-3개 짝 맞춰야 함)
- 표에서 헤더 아래 구분선(`|---|---|`) 빠뜨림
- 목록 하위 항목 들여쓰기 안 함 (스페이스 2칸 필요, 탭 X)
- `#` 뒤에 스페이스 안 넣음 (`#제목` X, `# 제목` O)
